How this works

Every certificate gets an ID

Issued the moment a student passes a course assessment, in the format MNL-YEAR-INITIALS-NUMBER.

02

The score comes with it

Verification shows the assessment score, not just a pass/fail — the same standard as the certificate itself.

03

Revoked stays visible

If a certificate is later revoked, this page says so rather than quietly returning nothing.
